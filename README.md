# Raheel Jamal - Portfolio

A professional portfolio website built with React and Framer Motion.

## 🚀 Quick Start (Local)

```bash
# Install dependencies
npm install

# Start development server
npm start
```

Visit `http://localhost:3000`

---

## 📦 Deploy on Vercel (Recommended - Free)

1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign up with GitHub
3. Click **"New Project"** → Import your repo
4. Click **Deploy** — Done! ✅

---

## 📦 Deploy on Netlify (Alternative - Free)

1. Run `npm run build` — creates a `build/` folder
2. Go to [netlify.com](https://netlify.com)
3. Drag & drop the `build/` folder → Done! ✅
